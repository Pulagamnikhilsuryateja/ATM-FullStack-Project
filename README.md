# ATM Full Stack Project

## 🚀 Tech Stack
- Java Spring Boot
- MySQL
- HTML, CSS, JavaScript

## 💡 Features
- User Login
- Check Balance
- Deposit Money
- Withdraw Money

## 🏗️ Project Structure
- backend → Spring Boot APIs
- frontend → UI (HTML/CSS/JS)

## ▶️ How to Run

### Backend
cd backend
mvn spring-boot:run

### Frontend
Open index.html in browser
